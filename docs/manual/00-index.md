# Welcome to TinyBooth Sound Studio

A native desktop recorder + visualizer + exporter for one-person bedroom-studio work. Opinionated by design — the tool nudges you toward a clean take with the recording-tone preset baked into every capture.

## What's in this manual

- **Welcome** — this page, plus a short *Getting started* walkthrough.
- **Reference** — one chapter per feature: Recording, Recording tones, Editing profiles, Projects, Export, Importing Suno stems, Self-update.
- **Using this manual** — how the manual itself works.
- **Appendix** — Troubleshooting and File formats.

## Quickest path to a recording

1. Plug in your mic or interface.
2. Open the **Record** tab.
3. Pick your input device. Leave the recording-tone preset on **Guitar** if you don't know which to choose; it's the gentlest one that still makes things sound nice.
4. Click ⏺. Make a sound. Click ⏹.

That's the loop. Everything else in this manual is depth on top of that loop.

## Philosophy in one sentence

The bedroom-studio opposite of a corporate DAW — one person, one mic, one take, sounding like it was lovingly recorded rather than left raw or over-produced.

## Where things live

| Thing | Location |
|---|---|
| Your settings | `%APPDATA%\TinyBooth Sound Studio\config.json` |
| Recording-tone profiles | `%APPDATA%\TinyBooth Sound Studio\profiles.json` |
| Default scratch projects | `%APPDATA%\TinyBooth Sound Studio\sessions\` |
| Your projects (when you pick a folder) | wherever you save them |

The manual itself is built into the binary — no install-path lookup, no broken doc links after an upgrade.
